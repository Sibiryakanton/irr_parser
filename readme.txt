IRR Parser - небольшой проект, выполненный для тестового задания. Цель задания: написать парсер по доске объявлений irr.ru, который позволял бы узнать самые активные часы и дни недели, в которые публикуется больше всего объявлений. Т.к. Время публикации сохраняется только для сегодняшнего дня, то и измерение суточной активности проводится только по нему, в связи с чем отчет рекомендуется запускать в 22-23 часа по времени выбранного региона.

Условие ТЗ:
1. На первой странице поле для ввода URL с сайта http://irr.ru/ , поле для ввода email и кнопка "получить отчет". Например, http://irr.ru/real-estate/apartments-sale .
2. Нажимаем на "Получить отчет", запускается таска Celery, которая проходит по всем страницам из URL.
3. В базу записываются дата и время размещения объявления.
4. После выполнения таски Celery на почту приходит URL, на который пользователь переходит и видит график "Популярного времени размещения" и "Популярный день недели".

Чтобы установить все необходимые пакеты, введите следующей команды:
pip install --upgrade pip
pip install -r requirements.txt

Также в UNIX-системах необходима следующая команда:
sudo apt-get install python3-tk
