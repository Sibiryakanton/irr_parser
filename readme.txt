IRR Parser - небольшой проект, выполненный для тестового задания. Цель задания: написать парсер по доске объявлений irr.ru, который позволял бы узнать самые активные часы и дни недели, в которые публикуется больше всего объявлений. Учитывайте, что irr.ru показывает точное время публикации только у объявлений последнего дня.

Условие ТЗ:
1. На первой странице поле для ввода URL с сайта http://irr.ru/ , поле для ввода email и кнопка "получить отчет". Например, http://irr.ru/real-estate/apartments-sale .
2. Нажимаем на "Получить отчет", запускается таска Celery, которая проходит по всем страницам из URL.
3. В базу записываются дата и время размещения объявления.
4. После выполнения таски Celery на почту приходит URL, на который пользователь переходит и видит график "Популярного времени размещения" и "Популярный день недели".


УСТАНОВКА

Проект написан для Django 1.11 под Python 3.5. Будем считать, что вы уже создали виртуальное окружение, чтобы ставить в него необходимые пакеты.

Установка зависимосетй делается в две команды:
pip install --upgrade pip
pip install -r requirements.txt

В UNIX-системах также понадобится установить еще один пакет для системы:
sudo apt-get install python3-tk
